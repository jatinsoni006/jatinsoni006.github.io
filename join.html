Join the Team

We would love to have you on board our team! Each year, we are tasked to build a robot depending on the game released by the FIRST Robotics Competition. We need all the minds we can get to make this happen. There are two sessions of operation, listed below.Joining the team is easy! Follow the instructions below.


Headstart

Headstart is a week long summer camp hosted at Monta Vista. We go through basic training and prototyping to allow kids to get an advantage during the Build Season. It is not required to sign up for Headstart if you want to participate during the Build Season.


Build Season

This is a 6-week session which starts after the game for the year has been unveiled. We work tirelessly to produce the best robot that we can during this time. Build Season consists of workshop days throughout the week where students can come participate in prototyping and building the final robot. You can see the build schedule here.
